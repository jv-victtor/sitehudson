# Hudson - Site exemplo (Django)

Projeto Django de exemplo com versões desktop e mobile do layout fornecido nas imagens anexas.

Como usar

1. Crie um ambiente virtual e instale dependências:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Rode as migrações e execute o servidor:

```powershell
python manage.py migrate
python manage.py runserver
```

3. Abra `http://127.0.0.1:8000/` no navegador.

Substitua a imagem placeholder em `static/img/hero.svg` pela foto real (use o arquivo fornecido nas imagens anexas) para obter o visual exato.
